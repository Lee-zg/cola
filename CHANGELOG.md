# Changelog

All notable changes to this project will be documented in this file.

This project follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html) and uses Conventional Commits for commit messages.

## [0.1.0] - 2026-05-13

### Added

- Initial Wails desktop application.
- Local SQLite bookmark storage.
- Bookmark CRUD, search, tags, folders, and rule-based analysis.
- Browser and Netscape HTML importers.
- Static HTML export and local read-only Web catalog.
- Backup and restore support.
- Vue 3 desktop management UI.
- Initial unit and integration tests.
