# Cola Bookmarks 设计规则

## 配色原则

Cola Bookmarks 使用低饱和暖色调，目标是降低长时间整理书签时的视觉疲劳，同时保留“可乐”相关的复古、醇厚、焦糖质感。界面应优先使用纯色和轻量层级，不使用高饱和渐变或大面积冷色。

## 颜色令牌

| 角色 | 名称 | HEX | RGB | 使用规则 |
| --- | --- | --- | --- | --- |
| Primary | 焦糖棕 Caramel | `#C28455` | `194, 132, 85` | 品牌、主按钮、进度条、当前选中菜单项、关键交互焦点。 |
| Accent Danger | 柔和灰红 Dusty Red | `#A95C58` | `169, 92, 88` | 警告、删除、危险操作、重要提醒。 |
| Accent Info | 雾霾灰蓝 Slate Blue | `#798C9F` | `121, 140, 159` | 标签、管理工具、次要信息高亮。 |
| Background | 燕麦白 Oatmeal Cream | `#F8F5F0` | `248, 245, 240` | App 全局底色。 |
| Surface | 纯净白 Pure White | `#FFFFFF` | `255, 255, 255` | 卡片、面板、列表容器、表单输入底色。 |
| Text Primary | 深焙咖啡色 | `#4A423C` | `74, 66, 60` | 标题、正文、用户输入内容。 |
| Text Secondary | 浅灰棕色 | `#8A817C` | `138, 129, 124` | URL、日期、副标题、占位说明。 |
| Text Disabled | 浅米灰色 | `#D3CCC6` | `211, 204, 198` | 禁用按钮、失效链接、不可用信息。 |

## 应用规则

- 背景固定使用 `#F8F5F0`，卡片和工具面板使用 `#FFFFFF`，通过轻微焦糖色阴影建立层级。
- 主操作按钮、当前导航项、输入聚焦边框使用焦糖棕 `#C28455`。
- 危险操作统一使用柔和灰红 `#A95C58`，不要混用高饱和红。
- 标签、分类、设置类辅助状态使用雾霾灰蓝 `#798C9F`。
- 文本禁止使用纯黑，主文本使用 `#4A423C`，次级文本使用 `#8A817C`。
- 渐变只允许作为极弱背景氛围；若必须使用，限制在 `#C28455` 与 `#D49A6E` 之间。
- 页面布局应保留足够留白，避免高密度控件贴边；卡片间距和行距优先保证可读性。

## CSS 变量映射

前端主题变量保持语义化，后续 UI 重构只能引用变量，不直接写死 HEX。

```css
--bg: #F8F5F0;
--panel: #FFFFFF;
--panel-strong: #4A423C;
--text: #4A423C;
--muted: #8A817C;
--disabled: #D3CCC6;
--line: #E6DDD4;
--line-strong: #D3CCC6;
--accent: #C28455;
--accent-strong: #A66D43;
--accent-soft: #F3E4D7;
--info: #798C9F;
--info-soft: #E5EBF0;
--danger: #A95C58;
--danger-soft: #F1DEDA;
--shadow: 0 18px 48px rgb(194 132 85 / 10%);
```
